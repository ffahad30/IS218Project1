About This Website

This website will teach you how to build an HTML 5 website. With this tutorial, you will learn to build your website using Bootstrap as the CSS framework. You will also learn how to use Git, Vi in Git, Docker, Web Servers (Apache and Nginx), Linux Commands, and Azure. Your website will be created using the following steps:

1. Using the command line, Linux commands, and Vi in Git Bash, create the HTML and CSS files for your website.

2. Push your HTML and CSS files from Git to Docker using the command "docker push (username)/(repository name)".

3. Build your image on Docker. Start by running the command "docker login azure". Once you have logged in, run the command "docker context create aci azureCloud". The context name in this command is azureCloud. Next, run the command "docker build -t (docker username)/(repository name) . ". This command builds your image. /p>

4. Run your image on Docker and deploy your Docker container to Azure. To do this, run the command "docker context use azureCloud". This ensures you are using the azureCloud context. Next, run "docker run -dp 80:80 (docker username)/(repository name)" to use Apache or "docker run -dp 80:80 (docker username)/(repository name) nginx" to use Nginx when deploying your image to Azure.

5. Using Git Bash, find your website's IP address. This can be done by running "docker ps". The IP Address will be under "Ports".

6. In a web browser, enter the IP address to access your website.
