What is Git?

Git is a Version Control System. It allows you to manage your projects through the use of terminals and repositories. You can create your projects using the terminals and store them in the repositories. Git also allows you to collaborate with others on projects through repositories. It is very versatile and makes building websites significantly easier. It is also an important resource used frequently in the professional world and is therefore an essential skill. Learn more here: https://git-scm.com/about.

Git Commands

  • git init - initializes Git Repository
  • git status - shows the status of the Git Repository
  • git add . - adds the files that have been changed
  • git commit -m "message" - commit changes and writes the description for what you accomplished with this commit
  • git checkout (name of branch) - switches to the named branch
  • git push origin (name of current branch) - pushes changes in current branch to the GitHub Repository
  • ls -a (name of folder) - shows all the hidden files and folders in specified folder
  • git checkout -b (name of new branch) - changes the branch to the new branch
  • git merge (name of branch) - moves files from the named branch into your current branch
  • git remote add origin (repository link) - connects GitHub repository to Git Bash project
  • git remote show origin - shows the origin (repository link)
  • git pull origin (name of branch) - pulls the named branch into the current branch
