Linux

What is Linux?

Linux is an operating system that allows you to manage your project directories and files. This is done through the use of Linux Commands. Linux is important to know because these commands are how you manage and navigate your project when only using the command line. It is an important skill for command line development and furthers your ability to function in a professional setting. Learn more about Linux here: https://www.linux.org/.

How to Use Linux

Run Linux commands in the command line of Git Bash.

Linux Commands

  • ls (name of directory) - list the contents of the directory
  • pwd - shows your present working directory
  • cd (name of directory) - changes directory
  • ~ - your home directory of the computer you're on
  • cd ~ - changes directory to your home directory
  • mkdir (name of new directory) - makes a directory
  • press tab key - autofills as much of the directory path as it can
  • touch - creates empty file
  • vi (name of file) - opens Vi word processor for named file
  • rm -r (name of directory) - removes directory
  • rm (name of file) - removes file
  • mv - move
  • mv (current file name) (new file name) - renames file
  • mv (current file name) (name of directory)/(new file name) - moves file into named directory and renames it
  • clear - clears the screen
