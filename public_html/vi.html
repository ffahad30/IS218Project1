Vi

What is Vi?

Vi is a text editor that allows you to edit your HTML and CSS files within Git Bash. It consists of both insert mode and command mode. Vi is essential to know because it is the main method for editing your project files while only using the command line. You can learn more about Vi here: https://www.vim.org/about.php.

How to Access Vi

In the command line of Git Bash, run the command "vi (name of file)". This will open the Vi text editor and allow you to make changes to your HTML and CSS files.

Vi Commands

  • vi (name of file) - opens text editor for named file
  • command mode - the default mode when you open Vi ; allows you to run commands
  • i - insert mode ; allows you to insert text
  • press escape key - removes you from insert mode and puts you back in command mode
  • :w - write the text in the file using command mode
  • :q - quit vi using command mode
  • :wq - writes and quits simultaneously
  • :q! - allows you to quit vi even if you haven't modified the file
